DOCUMENTAÇÃO iBIDEN

A CONSTRUÇÃO

Para que a ferramenta iBIDEN pudesse oferecer recursos relacionados com o Clima&Tempo e seu Impacto Ambiental, foi feito o uso de linguagens de programação e integração com APIs, que você pode ter acesso a mais informações nessa seção.

Para a construção da página WEB foi utilizado a linguagem de marcação de hipertexto HTML (HyperText Markup Language) para criar e estruturar conteúdo acerca das questões que giram em torno deste estudo. Ela foi fundamental para especificar a estrutura e o formato dos elementos da página, como textos, imagens, links e outros elementos interativos presentes na plataforma.

Além disso, também foi feito o uso do CSS (Cascading Style Sheets) que é uma tecnologia essencial que complementa o HTML na construção de páginas da web. O CSS pode ser definido como uma linguagem de estilo que separa a apresentação e o layout do conteúdo. Ele permitiu que fossem criadas regras de estilo para controlar a aparência visual dos elementos HTML, como cores, fontes, espaçamentos, tamanhos e posicionamento dos elementos presentes na plataforma. O CSS foi fundamental para a personalização e a estilização das páginas web, tornando-as visualmente atraentes e funcionais para os usuários.

Em relação à parte lógica da estrutura do motor de busca, foi realizada a ligação na API do Google para que ele fosse capaz de retornar os resultados das buscas feitas pelos usuários, levando em consideração os parâmetros de busca de resultados em português que contivessem as palavras chaves: "Dengue", "Aedes Aegypt", “Mudanças Climáticas", "Efeito Estufa", "Impacto Ambiental" e "Aquecimento Global”.
Isso permitiu que o mecanismo de busca aproveitasse a enorme base de dados do Google para buscar informações relevantes em tempo real, alinhando-se com os parâmetros de busca definidos. Para tanto, foi utilizada a linguagem de programação Javascript que desempenhou um papel crucial nesse processo.

JavaScript é uma linguagem versátil e amplamente suportada que é executada diretamente no navegador do usuário. Isso possibilitou a criação de uma experiência de busca interativa e responsiva, que permite aos usuários inserir suas consultas, ver os resultados e interagir com as informações de maneira fluida.

A integração da API do Google e o uso da linguagem de programação JavaScript desempenharam um papel crucial na criação de um motor de busca eficaz e amigável, que fornece aos usuários acesso rápido e preciso às informações desejadas sobre temas ambientais críticos.

Portanto, a partir das ferramentas supracitadas aplicadas ao sistema focado no sistema meteorológico e os impactos ambientais e suas consequências na saúde, foi possível criar um motor de busca de impactos ambientais que não apenas rastreie e identifique esses impactos e as torne acessíveis a todos os públicos.
O acesso à informação desempenha um papel fundamental na resolução de problemas, proporcionando uma base sólida para a tomada de decisões fundamentadas em dados concretos, reais e atualizados, o que, por sua vez, gera incentivos para a implementação de soluções sustentáveis.

Com base nisso, foi realizada a confecção de um motor de busca, como pode ser observado na Figura 1, sendo esta a primeira interface disponível denominada "Home", na qual o usuário tem a capacidade de inserir dados relacionados à cidade de interesse, com o objetivo de obter informações detalhadas sobre as condições meteorológicas locais.

Esses dados foram obtidos por meio da API Infodengue, um sistema inovador de alerta para arboviroses que combina fontes de dados diversas para gerar informações precisas e atualizadas sobre a situação epidemiológica.


FIGURA 9 - CASOS DE DENGUE NO BRASIL

INFODENGUE API

Além do recurso detalhado acima, com a integração estabelecida com a API Infodengue, foi desenvolvido um módulo específico para a previsão de casos de dengue no Brasil. Esse recurso evidencia a relação entre a incidência de dengue e as mudanças climáticas, já que as condições meteorológicas, como temperatura e umidade, impactam diretamente a proliferação do mosquito Aedes aegypti, transmissor da doença. Conforme aponta o Infodengue, “as variações climáticas afetam diretamente o ciclo de vida e a capacidade de disseminação do mosquito, tornando o monitoramento climático essencial para a previsão e controle de epidemias” (Infodengue, 2024).
A previsão de casos é apresentada em um gráfico que combina duas visualizações complementares: barras representando os casos reais registrados e uma linha que indica a projeção de casos futuros. Essa abordagem permite que o usuário visualize, de forma clara, as tendências passadas e futuras, possibilitando uma comparação direta entre os dados reais e os estimados. A integração desses elementos gráficos facilita a análise e torna o recurso acessível para gestores e profissionais de saúde, ajudando na identificação de períodos de maior risco e no planejamento de intervenções preventivas.

O QUE É A DENGUE?

Também está disponível uma cartilha dedicada exclusivamente à prevenção da dengue. Este recurso contém uma série de medidas práticas que os indivíduos podem adotar para reduzir o risco de contrair a doença e controlar a proliferação do mosquito transmissor. Desde dicas simples para eliminar criadouros de mosquitos até orientações sobre o uso de repelentes e a instalação de telas protetoras, a cartilha visa capacitar os usuários a adotarem atitudes proativas para proteger a si mesmos e suas comunidades contra a dengue.

A SUA SEGURANÇA

Para melhorar a usabilidade do sistema desenvolvido e orientar os usuários nas análises realizadas, foi criado um chatbot com foco em dengue e mudanças climáticas.
O seu objetivo é proporcionar uma navegação intuitiva e educativa pelo conteúdo do website, além de explicar as análises disponíveis no dashboard analítico, o que facilita o acesso a informações relevantes e promove uma melhor compreensão dos dados relacionados a esses temas.
Além disso, essa ferramenta foi configurada para reconhecer variações de palavras e diferentes grafias, o que facilita a interação, respondendo de maneira precisa e eficiente, independentemente do uso de letras maiúsculas, minúsculas ou de pequenas variações de termos. Isso permite que o chatbot seja um instrumento acessível para todos os tipos de usuários, desde os menos familiarizados com a terminologia de dengue e mudanças climáticas até os mais experientes, que podem rapidamente acessar páginas no site por meio dos links fornecidos.
